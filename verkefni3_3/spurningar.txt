1:Hver er munurinn á API og vefþjónustu (e. web services or Web API)? Hvað er sameiginlegt með þeim?

web servers er hugbúnaðarkerfi sem ætlað er að styðja samhæfð samskipti milli tölvu milli netkerfis 

og api er sett af skilgreiningum undirritunar, siðareglur og verkfæri til að byggja upp hugbúnað. 
Almennt er það sett af skýrt skilgreindum aðferðum við samskipti milli ýmissa hugbúnaðarhluta.
